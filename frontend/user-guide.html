1. Getting Started

Welcome to Chalo India Flight Booking Chatbot! This intelligent assistant helps you book flights between India and Australia.

How to Access

1 Open your web browser
2 Navigate to: http://localhost:3000/index.html
3 Click the orange chat icon in the bottom-right corner
4 Start chatting with the bot!

2. Booking a Flight

To book a regular flight, follow these steps:

Step-by-Step Booking Process

1 Start Booking: Type book or book flight
2 Enter Your Name: Provide your full name
3 Enter Email: Provide a valid email address (e.g., name@example.com)
4 Enter Mobile: Provide your mobile number with country code
5 Enter Passport: Provide your passport number (minimum 5 characters)
6 Select Departure Country: Type "India" or "Australia"
7 Select Departure City: Choose from the list (type number 1-10 or city name)
8 Select Destination Country: Type the opposite country
9 Select Destination City: Choose from the list
10 Enter Departure Date: Format: DD/MM/YYYY (e.g., 25/12/2024)
11 Enter Return Date: Type date or "no" for one-way trip
12 Enter Passengers: Number of passengers (1-9)
13 Select Travel Class: Type "Economy", "Business", or "First"
14 Confirm: Review summary and type confirm
Example Booking Flow:
User: book
Bot: Step 1: What is your full name?
User: John Doe
Bot: Step 2: What is your email address?
User: john@example.com
... (continues through all steps)

3. Viewing and Booking Deals

Viewing Available Deals

To see all available flight deals:

Type: deals or show deals

Searching for Specific Deals

You can search for deals by:

  • Route: deals from melbourne to delhi
  • Month: deals in december
  • Price: lowest fare deals
  • Combined: lowest fare deals in december for melbourne to delhi

Booking with a Deal ID

When you find a deal you like, book it using the Deal ID:

1 Type: book deal id 1 (replace 1 with the actual Deal ID)
2 The route is automatically set from the deal (no need to select cities!)
3 Follow the booking steps (name, email, mobile, passport, dates, passengers, class)
4 Confirm your booking
Example Deal Booking:
User: deals
Bot: Shows list of deals with Deal IDs
User: book deal id 3
Bot: Great! I'll help you book Deal ID 3. Route: MEL-DEL, Price: AUD $663...
User: John Doe
... (continues with simplified booking steps)

4. Available Commands

Booking Commands

  • book - Start flight booking
  • book flight - Start flight booking
  • book deal id X - Book a specific deal (X = Deal ID)

Deal Commands

  • deals - Show all available deals
  • show deals - Show all available deals
  • lowest fare deals - Show cheapest deals
  • deals from [city] to [city] - Search deals by route
  • deals in [month] - Search deals by month

Information Commands

  • help - Show help information
  • terms - View terms and conditions
  • contact - Contact support information
  • cancel - Cancellation policy
  • what is chalo india - Company information

General Commands

  • hi or hello - Greet the chatbot
  • who are you - Learn about the chatbot

5. Frequently Asked Questions

Q: How do I cancel a booking?

A: Type cancel to see the cancellation policy. For actual cancellations, contact support.

Q: Can I book for multiple passengers?

A: Yes! When asked for passengers, enter the number (1-9). The system will book for all passengers.

Q: What if a deal is out of stock?

A: The chatbot will inform you if a deal is unavailable. Choose another deal or book a regular flight.

Q: Can I modify my booking?

A: Currently, modifications require contacting support. Type contact for support information.

Q: What payment methods are accepted?

A: Payment details are provided after booking confirmation via email.

Q: How do I know if my booking was successful?

A: You'll receive a confirmation message and email after typing confirm.

6. Troubleshooting

Chatbot Not Responding

  • Check if the server is running (should see "Server is running" message)
  • Refresh the page
  • Check browser console for errors

Booking Steps Not Working

  • Make sure you're following the exact format requested
  • For dates, use DD/MM/YYYY format
  • For email, include @ and .
  • For passport, use at least 5 characters

Deals Not Showing

  • Try typing deals again
  • Check if deals are available (some may be out of stock)
  • Try a different search query

Error Messages

  • "Server error": Backend server may be down. Check server status.
  • "Invalid format": Check the format of your input (dates, email, etc.)
  • "Deal not found": The Deal ID doesn't exist. Check available deals first.
